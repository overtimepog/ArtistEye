# /art-critique

## When to Use
When you want expert artistic feedback on game art code, visual designs, or UI implementations.

## What It Does
1. Reviews the visual code/design holistically
2. Evaluates composition, color harmony, visual hierarchy
3. Checks style consistency and mood alignment
4. Verifies spatial correctness and accessibility
5. Provides prioritized, actionable improvements

## Agent Delegation
Delegates to **art-director** (Opus, READ-ONLY) for high-level artistic critique.

## Output Format
- Overall artistic impression
- Strengths of the current design
- Issues categorized by type and severity with file:line references
- Prioritized recommendations for maximum visual impact
